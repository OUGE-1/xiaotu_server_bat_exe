## 📖 项目简介

`server_bat.exe` 是一个将批处理命令封装为图形化/命令行工具的启动器，专为 **Minecraft 服务器**（或其他 Java 服务端）设计。  
你只需将你的服务器核心文件（如 `server.jar`）与程序放在一起，即可快速启动，并支持通过 **Lyt it 插件** 实现内网穿透/联机功能。
